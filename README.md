# Offensive Security Portfolio

This repository contains my practical materials related to penetration testing, including write-ups, tools, and professional report templates.

## Repository Structure

- **writeups/** — Detailed write-ups on vulnerabilities discovered during labs and real engagements (TryHackMe, HackTheBox, etc.).
- **tools/** — Custom scripts and utilities developed to automate various penetration testing tasks.
- **reports/** — Professional templates and example reports used to document findings for clients and employers.
